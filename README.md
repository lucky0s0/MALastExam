## ✨ 주요 기능
- 가위 / 바위 / 보 선택
- 컴퓨터 랜덤 선택
- 승 / 무 / 패 전적 누적
- 전적 초기화 기능

## ✨ 주요 기능
- 1 ~ 69 범위 랜덤 숫자 생성
- UP / DOWN 힌트 제공
- 시도 횟수 카운트
- 정답 시 축하 메시지 출력
